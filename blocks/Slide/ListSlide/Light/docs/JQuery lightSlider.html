JQuery lightSlider

JQuery lightSlider is a lightweight responsive Content slider with carousel thumbnails navigation

Demo Try it JsFiddleCodepen

Main Features

  • Fully responsive - will adapt to any device.
  • Separate settings per breakpoint.
  • Gallery mode to create an image slideshow with thumbnails
  • Supports swipe and mouseDrag
  • Add or remove slides dynamically.
  • Small file size, fully themed, simple to implement.
  • CSS transitions with jQuery fallback.
  • Full callback API and public methods.
  • Auto play and infinite loop to create a content carousel.
  • Keyboard, arrows and dots navigation.
  • Chrome, Safari, Firefox, Opera, IE7+, IOS, Android, windows phone.
  • Slide and Fade Effects.
  • Auto width, Vertical Slide, Adaptiveheight, Rtl support...
  • Multiple instances on one page.
  • Slide anything (youtube, vimeo , google map ...)

How to use lightSlider?

The Code

add the Following code to the <head> of your document.

  1. <link type="text/css" rel="stylesheet" href="css/lightslider.css" />                  
  2. <script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
  3. <script src="js/lightslider.js"></script>
  4. // Do not include both lightslider.js and lightslider.min.js

HTML Structure

  1. <ul id="lightSlider">
  2.   <li>
  3.       <h3>First Slide</h3>
  4.       <p>Lorem ipsum Cupidatat quis pariatur anim.</p>
  5.   </li>
  6.   <li>
  7.       <h3>Second Slide</h3>
  8.       <p>Lorem ipsum Excepteur amet adipisicing fugiat velit nisi.</p>
  9.   </li>
  10.   ...
  11. </ul>

Call lightSlider!

  1. <script type="text/javascript">
  2.   $(document).ready(function() {
  3.     $("#lightSlider").lightSlider(); 
  4.   });
  5. </script>

Public methods

  1. <script type="text/javascript">
  2.   $(document).ready(function() {
  3.     var slider = $("#lightSlider").lightSlider();
  4.     slider.goToSlide(3);
  5.     slider.goToPrevSlide();
  6.     slider.goToNextSlide();
  7.     slider.getCurrentSlideCount();
  8.     slider.refresh();
  9.     slider.play(); 
  10.     slider.pause();    
  11.   });
  12. </script>
